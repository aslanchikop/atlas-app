{% if lang == 'en' %}What is ATLAS?{% else %}ATLAS дегеніміз не?{% endif %}

{% if lang == 'en' %} ATLAS (Autonomous Terrestrial Life Analysis System) is an intelligent research platform that queries the NASA Exoplanet Archive in real time, applies machine-learning habitability models, and generates scientific insights — all within a beautiful, interactive interface. Built as a diploma project to demonstrate the fusion of data science, astrophysics, and modern web design. {% else %} ATLAS (Автономды Жердегі Тіршілікті Талдау Жүйесі) — NASA Экзопланета мұрағатынан нақты уақытта деректерді сұрайтын, машиналық оқуға негізделген тіршілік қабілеттілігі үлгілерін қолданатын және ғылыми түсініктер беретін зерттеу платформасы. Деректер ғылымы, астрофизика және заманауи веб-дизайнның синтезін көрсету үшін диплом жобасы ретінде жасалды. {% endif %}
