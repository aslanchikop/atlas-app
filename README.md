## 🌌 Жоба туралы (Project Overview)

**ATLAS** — NASA-ның Экзопланеталар мұрағатынан (NASA Exoplanet Archive) нақты уақытта деректерді сұрайтын, машиналық оқу және математикалық модельдер негізінде планеталардың тіршілікке жарамдылығын (Habitability) бағалайтын, сондай-ақ жұлдызды жүйелерді интерактивті 3D форматта визуалдайтын зияткерлік веб-платформа.

Бұл жүйе заманауи **Bento Grid** және **Glassmorphism** дизайн концепцияларымен, бөлшектер жүйесімен (particle systems) және озық жасанды интеллект көмекшілерімен жабдықталған. Жоба ғылыми деректерді талдау (Data Science), астрофизика және заманауи веб-технологиялардың тоғысқан нүктесі ретінде әзірленген.

## 🌟 Негізгі модульдер мен мүмкіндіктер (Key Features)

### 1. 🔭 Экзопланеталарды зерттеу (Explore & TAP API)
* NASA TAP API арқылы нақты уақытта белсенді ғарыш миссияларынан (Kepler, TESS, K2 және т.б.) жұлдыздар мен планеталар туралы мәліметтерді тартады.
* Планетаның атмосфералық құрамын, магниттік өрісін, серіктер санын және ықтимал қауіп-қатерлерін (hazards) болжайтын эвристикалық алгоритм.

### 2. 🪐 Интерактивті 3D Орбиталық Визуализатор
* **Plotly.js** негізінде жасалған 3D кеңістік моделі.
* Пайдаланушы жұлдыз маңындағы планеталардың нақты орбиталық қашықтықтары мен жылдамдықтарын интерактивті түрде айналдырып, зерттей алады.
* Тіршілік аймағының (Habitable Zone) шекаралары жасыл түспен ерекшеленіп көрсетіледі.

### 3. 📊 Салыстырмалы Аналитика (Compare Module)
* Бірнеше экзопланетаны таңдап, олардың радиусын, массасын, температурасын, ESI көрсеткішін және тартылыс күшін радарлық (Radar) және бағандық диаграммалар арқылы салыстыру.

### 4. 🚀 Релятивистік саяхат симуляторы (Relativistic Journey)
* Саяхат жылдамдығын (жарық жылдамдығының пайызымен, $c$) таңдау мүмкіндігі.
* Эйнштейннің Салыстырмалылық теориясына негізделген уақыттың баяулауын (**Time Dilation**) және ғарышкерлер үшін де, Жердегі бақылаушылар үшін де ұшу уақытын есептеу.

### 5. 🤖 Қос ЖИ Көмекшісі (Dual AI System)
* **ATLAS AI:** Жалпы ғылыми сұрақтар мен деректер талдауына жауап беретін интеллектуалды чат.
* **Laika 🐾 (Ғарыш Mascot-ы):** 1957 жылы ғарышқа ұшқан тұңғыш жер тұрғыны Лайка иттің рөліндегі интерактивті, эмоционалды әрі көңілді ғарыш көмекшісі. Ол пайдаланушының іс-әрекеттеріне қарай эмоциясын өзгертіп отырады.

### 6. 🏆 Геймификация және Жетістіктер жүйесі (Achievements)
* Пайдаланушының белсенділігін арттыруға арналған 14 бірегей ғылыми жетістіктер жиынтығы (мыс. *First Contact*, *Earth Twin*, *Giant Hunter*).

## 📐 Ғылыми есептеулер мен алгоритмдер (Scientific Calculations)

Платформада қолданылатын негізгі физикалық формулалар:

### 1. Жерге ұқсастық индексі (Earth Similarity Index - ESI)
ESI планетаның өлшемі мен температурасы бойынша Жерге қаншалықты жақын екенін көрсетеді ($0$-ден $1$-ге дейінгі шкала):

$$ESI = \sqrt{ESI_R \cdot ESI_T}$$

Мұндағы өлшемдік ұқсастық ($ESI_R$) және температуралық ұқсастық ($ESI_T$):

$$ESI_R = \left(1 - \left|\frac{R_p - 1}{R_p + 1}\right|\right)^{0.57}$$

$$ESI_T = \left(1 - \left|\frac{T_p - 288}{T_p + 288}\right|\right)^{5.58}$$

*Ескерту:* $R_p$ — планета радиусы ($R_\oplus$), $T_p$ — беткі температурасы ($K$).

### 2. Жұлдыз жарықтылығы (Stellar Luminosity - L)
Күннің жарықтылығына қатысты есептеледі:

$$L = R_*^2 \cdot \left(\frac{T_{eff}}{5778}\right)^4$$

### 3. Мекендеуге жарамды аймақ шекаралары (Habitable Zone - HZ)
Судың сұйық күйде сақталуына мүмкіндік беретін жұлдызға дейінгі қашықтық (Астрономиялық бірлікпен, $AU$):

$$d_{inner} = 0.75 \cdot \sqrt{L}$$

$$d_{outer} = 1.77 \cdot \sqrt{L}$$

### 4. Релятивистік уақыт баяулауы (Time Dilation)
Лоренц факторы арқылы ғарышкерлер үшін баяулайтын уақытты есептейді:

$$\Delta t' = \Delta t \cdot \sqrt{1 - \frac{v^2}{c^2}}$$

Мұндағы $v$ — кеме жылдамдығы, $c$ — жарық жылдамдығы, $\Delta t$ — Жер уақыты.

## 🛠️ Технологиялық стек (Tech Stack)

### Бекэнд (Backend):
* **Python 3.14** — негізгі бағдарламалау тілі.
* **Flask 3.1** — жылдам, микро-веб фреймворк.
* **Scikit-learn / NumPy / Joblib** — тіршілік ықтималдығын болжайтын машиналық оқу моделін жүктеу және есептеу.
* **python-dotenv** — құпия кілттерді басқару.

### Фронтэнд (Frontend):
* **HTML5 & Vanilla CSS3** — озық Glassmorphism, Bento Grid және динамикалық CSS трансформациялары.
* **Vanilla JavaScript (ES6)** — ешқандай ауыр фреймворксіз жылдам реактивті өзара әрекеттесу.
* **Plotly.js** — жұлдыздық жүйелердің интерактивті 3D модельдерін құру.
* **Canvas API** — басты беттегі динамикалық ғарыш бөлшектері (starfield particle system).

### AI & Сақтау (AI & Storage):
* **Google Gemini 2.0 Flash** — негізгі аналитикалық және сұхбаттық ЖИ қозғалтқышы.
* **Локалды кэштеу** (`gemini_cache.json`) — ЖИ жауап беру жылдамдығын арттыру және API лимиттерін үнемдеу.
* **Session Storage** — пайдаланушының прогресі мен ашылған жетістіктерін сақтау.

## ⚙️ Орнату және локалды іске қосу (Installation)

Жобаны жергілікті компьютерде іске қосу үшін келесі қадамдарды орындаңыз:

1. **Репозиторийді клондаңыз:**
   ```bash
   git clone https://github.com/aslanchikop/atlas-app.git
   cd atlas-app
   ```

2. **Виртуалды ортаны құрып, белсендіріңіз:**
   ```bash
   python -m venv venv
   # Windows үшін:
   venv\Scripts\activate
   # macOS/Linux үшін:
   source venv/bin/activate
   ```

3. **Тәуелділіктерді орнатыңыз:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Қоршаған орта айнымалыларын теңшеңіз (.env):**
   Жобаның түбірлік каталогында `.env` файлын жасап, келесі мәндерді жазыңыз:
   ```env
   GEMINI_API_KEY=сіздің_gemini_api_кілтіңіз
   PORT=5000
   ```

5. **Серверді іске қосыңыз:**
   ```bash
   python app.py
   ```
   Браузерді ашып, [http://localhost:5000](http://localhost:5000) сілтемесіне өтіңіз.

## ☁️ Деплой және Өндіріс (Deployment)

Жоба **Railway** немесе **Heroku** платформасына бірден жүктелуге дайын. Өндірісте Flask қосымшасын жоғары жылдамдықпен ұстап тұру үшін **Gunicorn** веб-сервері қолданылады (`Procfile` арқылы жүзеге асады):

```procfile
web: gunicorn app:app --bind 0.0.0.0:$PORT --workers 2 --timeout 120
```
